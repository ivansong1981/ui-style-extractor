# Launch Announcement

## GitHub / Long Form

Open-sourcing `ui-style-extractor`.

It extracts a website's design system into reusable Markdown style guides for AI-generated pages.

The point is not to clone a screenshot. The point is to extract the actual visual rules behind a site so Codex, Claude Code, or another LLM can keep building new pages that still feel native to that system.

This first public version includes:

- a shared skill for Codex and Claude Code
- a reusable style-guide template
- two public examples: `99percent` and `stripe`
- validation demos and screenshots
- English and Chinese README
